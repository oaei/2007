You can also download all manually labeled data. If you want to use these data, please let me know about it (svabo at vse dot cz) and if you want to refer these data, please cite them as whole dataset. This dataset was firstly presented at ISWC-2005 as poster (see at OntoFarm project. Data are comma-separated; there are 5 columns:
  • name1 refers to label of concept1,
  • name2 refers to label of concept2,
  • result refers to manually labeled category for particular mapping,
  • onto1 refers to ontology (the order is given in table) from which concept is,
  • onto2 refers to ontology (the order is given in table) from which concept is,
In the column 'result', there are number of diverse categories:
  • + represents correct mapping,
  • i represents 'interesting' correct mapping,
  • / represents 'interesting' incorrect mapping,
  • - represents incorrect mapping,
  • s represents subsumption relation,
  • d represents that concepts are siblings,
  • n represents inversion roles,
  • ? represents unclear mapping,
  • r represents mapping between property and class, or vice versa,
  • t represents mapping that is trivial - regarding textual labels.
